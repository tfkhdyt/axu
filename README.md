# Arch linux eXplicit Updates

`axu` is a cli tool to check your Arch Linux explicitly installed packages updates. Inspired by [@antfu/taze](https://github.com/antfu/taze), but for Arch Linux

## Getting started

### Pre-requisites

- Arch Linux
- `pacman-contrib`
- `yay`
- `gawk`
- `coreutils`
- Rust (make deps)

### Installation

#### AUR

```bash
yay -S axu
```

### Usage

```bash
axu
```
